### 定义
**运维应用**是SREWorks基于云原生解决运维需求的应用软件。

<a name="fZj9i"></a>

### 组成
SREWorks提供完整的**运维应用**开发套件，包含

- 前端开发: 配置化前端模块开发
- 后端开发: 云原生构建/部署/发布
